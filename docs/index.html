a2000 (or a2k)

Welcome to a2000 (or a2k for short).

a2000 is a UI library design to capture the feel of the early web using modern web practices.

Note: This is a work in progress and is subjected to change. I welcome any and all feedback!

Check out the demo site for the full a2000 experience. You can also check out the Storybook to play around with each component. If you want to use a2000 in your own projects, you can follow the in

Styles

A2k comes with its own default set of styles

You can add it to your HTML using the CDN, like so:

<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/@a2000/styles/a2k-styles.css" />

Adding this stylesheet applies the default base styles, variables, cursors, and typefaces.

a2000 uses the Pixelated MS Sans Serif typeface
